Why?

Because some friends and I spent an hour sitting in Mumble trying to figure out what to play one night, and Steam inexplicably doesn't have this functionality built in.

There are some similiar features on other sites but none of them (that I found anyway) worked the way I wanted.

What does 「オマカセ」 mean?

"OMAKASE" (pronounced "oh-mah-kah-say") means something like "I'll leave it up to you". It's typically used in sushi restaurants to let the chef decide what sushi to serve. Seemed appropriate for the original planned functionality which did not include the list of shared games, only the "Just pick one" button.

It's written with Katakana because it looks cooler and I don't know the Kanji for it.

How does this work?

The core bits are smiley/steamapi for getting user/friend/owned-games data and the Steam storefront API for getting data about the games themselves. Then it's a simple set intersection after filtering by platform.

Are you gonna hack/steal my account?

That's not how it works. That's not how any of this works. Your Steam account ID (the big long number) is publicly available, as is the rest of the data the app shows. Note that your Steam login username is not necessarily the same as your vanity URL and there's no way to get your login username (what would be important for trying to steal accounts) from the account ID or vanity url token.

Only publicly available information is used, so you could do everything the app does by hand and a few sheets of paper (it would just be very time consuming).

It should let me include people from groups that I'm in!

It should! But the steamapi module didn't have a quick/easy way to get the name of the group or the list of members, plus it could potentially significantly increase the amount of work in the intersection request so it was left out for now. Maybe it will be added in the future.

It will probably never allow the intersection of arbitrary collections of people (i.e. people you're not friends with or in a group with).

Cool! There are probably a few of them since this was thrown together in a few hours. You can report it on GitHub.
